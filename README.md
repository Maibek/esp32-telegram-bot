# ESP32 Telegram Bot — Умный контроллер климата для террариумов

## О проекте
Это полноценная автоматическая система контроля микроклимата для двух независимых зон (например, муравейников Messor structor и Nicobarensis или двух террариумов).
ESP32 управляет подогревом, освещением, мониторит температуру и влажность через два датчика HTU21DF и отправляет все данные + тревоги прямо в Telegram.
Основные возможности

## Возможности
- Два независимых канала контроля климата  
- Автоматическое включение/выключение подогрева по заданным порогам  
- Управление освещением (одна группа на два канала)  
- Тревожные уведомления при:
  - высокой / низкой температуре
  - высокой / низкой влажности
  - неисправности датчика
- Настраиваемые интервалы повторных оповещений  
- Сохранение настроек в LittleFS
- Удобное кнопочное меню в Telegram  
- Watchdog, авторестарт при потере Wi-Fi, логирование причины перезагрузки  
- Защита от спама уведомлений (throttler)

## Telegram-бот с удобным интерфейсом
### Кнопочное меню:
Статус системы (RSSI, память, аптайм, температура ядра)
Датчики (отдельные карточки для каждой зоны)
Управление (переключение реле и света прямо из чата)
Настройки оповещений + сброс к дефолту

Поддержка JSON-команд (для будущей интеграции с другими сервисами)
Защита от дубликатов сообщений

## Используемое железо

### Аппаратные требования

| Компонент               | Кол-во | Примечание                              |
|-------------------------|--------|-----------------------------------------|
| ESP32 DevKit            | 1      | любая версия с 2×I²C                    |
| HTU21DF / SHT21         | 2      | датчики температуры + влажности         |
| Реле-модуль 5 В         | 1–2    | 2-канальный или два одноканальных       |
| Источник питания 5 В    | 1      | стабилизированный, ≥1 А                 |
| Резисторы подтяжки 4.7k | 4      | для каждой линии I²C (если нужно)       |

### Используемые пины (по умолчанию)

| Назначение         | Пин ESP32 | Примечание                     |
|--------------------|-----------|--------------------------------|
| SDA0 (датчик 0)    | GPIO 23   | I²C0                           |
| SCL0 (датчик 0)    | GPIO 22   | I²C0                           |
| SDA1 (датчик 1)    | GPIO 19   | I²C1                           |
| SCL1 (датчик 1)    | GPIO 18   | I²C1                           |
| Реле 0 (подогрев)  | GPIO 16   | LOW = включено                 |
| Реле 1 (подогрев)  | GPIO 17   | LOW = включено                 |
| Свет 0             | GPIO 2    | LOW = включено                 |
| Свет 1             | GPIO 4    | LOW = включено                 |

Питание: рекомендуется 5 В стабилизированное
